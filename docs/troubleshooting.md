# 🛠️ 트러블슈팅 보고서 (Troubleshooting Report)

AWS 인프라 구축 및 설정 과정에서 발생했던 문제 상황과 해결 과정을 기록한 문서입니다.

| 항목 | 내용 |
| :--- | :--- |
| **증상(문제 상황)** | AWS 계정 생성 및 프리티어 사용을 위한 결제수단(신용카드/체크카드) 등록 불가 |
| **원인 가설** | 특정 카드 브랜드(UnionPay 등)의 일시적인 해외 결제 차단 혹은 AWS와의 호환성 이슈 가능성 |
| **검증 방법** | 타 카드사 결제 실패 사례 및 인터넷상의 유사 AWS 결제 수단 등록 실패 이슈 검색, AWS 결제 시스템 지원 사양 확인 |
| **조치 내용** | 해외 결제 및 AWS 등록이 안정적으로 지원되는 다른 카드 브랜드(VISA 또는 MasterCard)의 카드를 새로 발급 및 등록 시도 |
| **결과** | 신규 등록한 카드를 통해 본인 인증 및 결제 수단 등록 성공, 프리티어 계정 활성화 완료 |
| **재발 방지** | 향후 클라우드 플랫폼 결제 수단 등록 시, 호환성이 높은 글로벌 결제 브랜드(VISA/MasterCard) 카드를 우선적으로 준비하고 해외 결제 차단 여부를 사전에 확인하도록 가이드 마련 |
